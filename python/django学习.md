## 相关命令

* 输出django版本

  ```shell
  python -m django --version
  ```

* django创建项目

  ```shell
  django-admin startproject mysite(项目名)
  ```

* 起服务

  ```shell
  python manage.py runserver <port>
  ```

* 新建应用

  ```
  python manage.py startapp <name>
  ```
